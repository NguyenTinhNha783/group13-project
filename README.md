# Group 13 Project

## Mô tả
Website quản lý user, bao gồm backend Node.js + MongoDB, frontend React.

## Công nghệ
- Node.js, Express
- MongoDB Atlas
- React
- Axios, Cors

## Hướng dẫn chạy
1. Backend:
   cd backend
   npm install
   npm run dev
2. Frontend:
   cd frontend
   npm install
   npm start

## Thành viên
- Nhã Nguyễn: Backend API
- Khang Lâm: Database MongoDB
- Vinh Từ: Frontend React
